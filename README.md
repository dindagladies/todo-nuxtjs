# Description

Todo Application using Nuxt JS. Also using [todo-api](https://github.com/dindagladies/todo-api)

## Build Setup

Install dependencies

```bash
npm install
```

Setup port axios, open nuxt.config.js then update this code base on your API url and port
```javascript
axios: {
    baseUrl: 'http://localhost:8000'
  },
```

## Usage

```bash
npm run dev
```
then run [http://localhost:3000/]() in your browser
